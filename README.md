# Site Builder

Site Builder is an AI-powered website generation platform that enables users to create, customize, and deploy modern websites with intelligent workflows and scalable full-stack architecture. The platform is designed to simplify website creation using reusable components, dynamic layouts, and AI-assisted customization.

---

## Features

### User Features

* AI-assisted website generation
* Create and customize modern responsive websites
* Dynamic reusable UI components
* Real-time preview and editing
* Responsive design support
* Authentication and user management
* Easy deployment workflow

### AI Features

* Intelligent website layout generation
* AI-powered content and component suggestions
* Automated design customization workflows
* Smart template generation

### Backend Features

* Scalable REST API architecture
* Secure authentication and authorization
* Database integration for storing projects
* Efficient content and component management
* Modular backend structure

## Installation & Setup

### Clone Repository

```bash
git clone https://github.com/jayz24-ship/site-builder.git
cd site-builder
```

---

## Backend Setup

```bash
cd backend
npm install
```

Create `.env` file:

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
OPENAI_API_KEY=your_openai_key
PORT=5000
```

Run backend:

```bash
npm run dev
```

---

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
